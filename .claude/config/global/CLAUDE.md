# Global Claude Infrastructure

These rules apply to every Claude Code session on this machine. Project-specific
conventions belong in each project's own CLAUDE.md and override anything here on
conflict.

## Source of truth

The rules below are imported from the claude-infra repo via symlink:
`~/.claude/rules/` -> `~/Desktop/Development/projects-active/new-claude-infra/core/rules/`

Edit the files in the repo, not in `~/.claude/`. Changes propagate immediately.
